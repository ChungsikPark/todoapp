# todoapp
react-native & redux & saga

## 요구사항

- todo item 을 클릭시 상세보기 페이지로 넘어감
- todo item 의 내용이 사용 가능한 컨테이너의 최대 영역을 넘어가면 ... 표시
- react native redux , saga 아키텍처를 적용해서 구현
- saga function list

  1. createTodo
  2. updateTodo
  3. deleteTodo
  4. getTodo

- 위 각각에 대한 saga function 에 대한 test code 작성 ( jest )
